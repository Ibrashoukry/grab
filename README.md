## `grab` - Command Line Tool
`grab` is a lightweight command line tool built in Rust that allows you to search for a pattern in your files.

### Usage
`grab [OPTIONS] <PATTERN> <PATH>`\
\
**Options:**\
`-h` or `--help`: Provides information about tool usage\
`-V` or `--version`: Outputs the release version\
`-i` or `--case-insensitive`: Allows searching for pattern to be case insensitive\
`-n` or `--line-number`: Outputs the line number of the lines that contain the pattern\
